# Algoritmo para Knight Tour

A ideia desse projeto é implementar uma solução para o problema do cavalo no tabuleiro de xadrez. A proposta é que dada uma casa inicial qualquer, o cavalo percorra todo o tabuleiro (8x8) sem repetir nenhuma posição.

Para implementar o projeto utilizei TDD, WPF e Xamarin.

**Requisitos para Instalação**

 - VisualStudio preparado para WPF e Xamarin.
